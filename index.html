The Game MD

The Game MD Website
Website Analysis
Desktop View

I like how they implemented a multi-column layout without making the content overwhelming to read. There is no more than 3 columns at a time.

I am not a fan of the small font on the navigation links. These links are very small and can be difficult to read for some users. I can also live without background-images where I have to scroll to see the content I'm looking for.

Mobile View(s)

I like the single column layout on this mobile view. Reading the services provided is much easier to follow when viewing on mobile. The navigation is much more accessible on the mobile view than the desktop view.

I don't like the image on the landing screen as I find it is kind of janky when you visit the site on a tablet. At some resoltions the image is scrollable but the focal point is never visible, making it obtrusive for no reason.

Rating: 6.5/10

Game Cycle

Game Cycle Website
Website Analysis
Desktop View

I like the simplicity of the design. Keeping to a 3 column maximum throughout the site. There is a reasonable amount padding around text, and the pictures and text don't take away from each other.

I do no like how hard it is to find the navigation links at the top of this page. Having white text and a transparent background colour makes the navigation difficult to read.

Mobile View(s)

The only thing I can say I'm a fan of on this mobile view is the navigation is always easily visible, as opposed to the Desktop view.

The 'Learn More' button is the colour as the its background image, making it very hard to spot on the mobile view. The header at the top of the page is also in plain white text while there is a slideshow of images as its background. I find this very hard to read, and other users will too.

Rating: 7.25/10

Liquidation Center

Lquidation Center Website
Website Analysis
Desktop View

The desktop view has more of an e-commerce feel to it which I really enjoy. It was very easy to navigate the different types of items, as well as the items added to my cart.

I don't like how the navigation switches from the left to the right side depending on what page you are on. The account settings is also very difficult to find, and many desktop viewers may not know it is there.

Mobile View(s)

The mobile view sacrifices some content on the mobile view (e.g. items for sale on the homepage). I like how the account settings is more clearly visible with its own icon in the navigation.

The hamburger menu is essentially useless on the mobile view as there are no revealed links or anything to guide you. Just a search bar. I think this is better accomplished with the other present search bar that has a select list alongside it.

Rating: 8.13/10
