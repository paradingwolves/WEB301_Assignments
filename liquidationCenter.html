Liquidation Center

The Liquidation Center implemented adaptive design for their website. Their navigation on the left side of the page turns into a hamburger menu, and the "featured items" are now the only items shown to the user on the homepage when the screens viewport is less than 767px wide. The images that are consistent with the desktop and the mobile view are replaced with smaller versions of the same image.

Desktop View

The desktop view has more of an e-commerce feel to it which I really enjoy. It was very easy to navigate the different types of items, as well as the items added to my cart. I don't like how the navigation switches from the left to the right side depending on what page you are on. The account settings is also very difficult to find, and many desktop viewers may not know it is there.

Mobile View(s)

The mobile view sacrifices some content on the mobile view (e.g. items for sale on the homepage). I like how the account settings is more clearly visible with its own icon in the navigation. The hamburger menu is essentially useless on the mobile view as there are no revealed links or anything to guide you. Just a search bar. I think this is better accomplished with the other present search bar that has a select list alongside it.

I would add more links to the hamburger menu if this was my website. The empitness and confusing search bar make it a very confusing feature the way it is.


Overall Rating: 8.13/10
